Introduction

Welcome to the Node.js technical documentation. This guide will help you understand and use Node.js effectively.

Node.js is a powerful runtime that allows you to execute JavaScript on the server-side.

What is Node.js?

Node.js is an open-source, cross-platform JavaScript runtime environment that executes JavaScript code outside a web browser.

  • It uses an event-driven, non-blocking I/O model
  • It's lightweight and efficient
  • It has a large ecosystem of open-source libraries
Installation

To install Node.js, follow these steps:

  1. Visit the official Node.js website
  2. Download the installer for your operating system
  3. Run the installer and follow the prompts

You can verify the installation by running:

node --version
Hello World

Let's create a simple "Hello World" program in Node.js:

console.log('Hello World!');

Save this in a file named hello.js and run it using:

node hello.js
Modules

Node.js uses a module system to organize and reuse code. Here's a simple example:

// In greet.js module.exports = function(name) { return `Hello, ${name}!`; }; // In main.js const greet = require('./greet'); console.log(greet('Node.js'));

This demonstrates how to create and use a custom module in Node.js.

NPM

NPM (Node Package Manager) is the default package manager for Node.js. It allows you to install and manage dependencies for your projects.

To initialize a new Node.js project with NPM:

npm init -y

To install a package:

npm install package-name
File System

Node.js provides a built-in module for interacting with the file system. Here's an example of reading a file:

const fs = require('fs'); fs.readFile('example.txt', 'utf8', (err, data) => { if (err) throw err; console.log(data); });
HTTP Module

The HTTP module allows Node.js to transfer data over HTTP. Here's a simple HTTP server:

const http = require('http'); const server = http.createServer((req, res) => { res.writeHead(200, {'Content-Type': 'text/plain'}); res.end('Hello World\n'); }); server.listen(8080);
Asynchronous Programming

Node.js uses an event-driven, non-blocking I/O model. This allows for efficient handling of concurrent operations.

Here's an example using Promises:

function fetchData() { return new Promise((resolve, reject) => { setTimeout(() => resolve('Data fetched'), 2000); }); } fetchData().then(data => console.log(data));
Events

Node.js has a built-in events module for handling and emitting events:

const EventEmitter = require('events'); class MyEmitter extends EventEmitter {} const myEmitter = new MyEmitter(); myEmitter.on('event', () => { console.log('An event occurred!'); }); myEmitter.emit('event');
